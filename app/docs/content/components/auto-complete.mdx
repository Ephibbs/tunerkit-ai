## Installation

First, ensure you have `backless-ai` installed:

```bash
npm install backless-ai
```

## Setup

To use the Chat Component, import it from the library and integrate it into your application:

```javascript
import { ChatComponent } from 'backless-ai';

function App() {
  return (
    <div className="App">
      <ChatComponent apiKey="YOUR_API_KEY" />
    </div>
  );
}

export default App;
```

### Configuration Options

You can configure the Chat Component using the following props:

- **`apiKey`**: Your Backless AI API key.
- **`theme`**: Customize the colors and fonts to match your brand.
- **`onMessageSend`**: Function that triggers when a user sends a message.
- **`initialMessages`**: Preload the chat with a set of messages.

Example with custom options:

```javascript
<ChatComponent
  apiKey="YOUR_API_KEY"
  theme={{
    backgroundColor: '#f0f0f0',
    color: '#333'
  }}
  onMessageSend={(message) => console.log("Message sent:", message)}
  initialMessages={[
    { author: "ai", content: "Hello! How can I assist you today?" }
  ]}
/>
```

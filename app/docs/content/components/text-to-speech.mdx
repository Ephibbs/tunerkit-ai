## Installation

Ensure that you have `backless-ai` installed:

```bash
npm install backless-ai
```

## Setup

To use the Text to Speech functionality, import the necessary functions from the library and integrate them into your application:

```javascript
import { getPublicClient } from 'backless-ai';

const client = getPublicClient('YOUR_API_KEY');
```

## Configuration Options

You can configure the Text to Speech function with the following parameters:

- **`input`**: The text string you want to convert to speech.
- **`model`**: Specify the TTS model to use.
- **`voice`**: Choose the voice type (e.g., 'alloy').

## Best Practices

- **Contextual Use**: Utilize TTS where it enhances the user experience, such as in accessibility features or interactive storytelling.
- **Performance Considerations**: Be mindful of the loading times and resource usage when integrating TTS, especially on mobile devices.
- **Privacy Compliance**: Ensure that any text processed by TTS does not violate user privacy or data regulations.
